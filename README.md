Tema: Desarrollo de un Generador de Contraseñas
Objetivo

Desarrollar un generador de contraseñas utilizando el lenguaje de programación Python, aplicando los conocimientos adquiridos en la asignatura, como la resolución de problemas, el manejo de datos, algoritmos, diagramas de flujo, estructuras condicionales, bucles, estructuras de datos y funciones, con el fin de fortalecer las habilidades de programación y comprender la importancia de crear contraseñas seguras para proteger la información.

Introducción

Hoy en día, la seguridad en internet es muy importante porque mucha gente utiliza plataformas digitales y almacena información personal en la red. Una de las mejores formas de protegerse es utilizar contraseñas seguras, ya que esto ayuda a evitar que personas no autorizadas accedan a nuestras cuentas y sistemas. Sin embargo, mucha gente elige contraseñas fáciles de adivinar, lo que aumenta el riesgo de sufrir ataques informáticos.
Con el fin de aplicar lo que se ha aprendido, se creó un programa para generar contraseñas utilizando el lenguaje de programación Python. Este proyecto permitió combinar conceptos básicos de programación, como algoritmos y manejo de datos, y dio como resultado una aplicación que funciona correctamente y crea contraseñas aleatorias y seguras según lo que el usuario prefiera.

Descripción del problema

Hay un problema común en la seguridad digital. Es cuando creamos contraseñas que no son seguras. A veces usamos nombres, fechas de nacimiento o números simples. Esto es un error porque estas contraseñas son fáciles de adivinar. Alguien podría usar un ataque de fuerza bruta o información personal para descubrirlas.
Para solucionar este problema, se creó un programa especial. Este programa genera contraseñas aleatorias con diferentes características. El usuario puede elegir la longitud de la contraseña y qué tipo de caracteres incluir. Puede elegir si quiere letras mayúsculas, letras minúsculas, números y caracteres especiales. De esta manera, creamos contraseñas más fuertes y difíciles de descubrir.

Relación de los contenidos de la asignatura

Durante el desarrollo del proyecto, se aplicarán los siguientes contenidos estudiados en la asignatura.
El proyecto comenzó con el análisis del problema y la identificación de los requisitos necesarios para construir una solución. Luego, se utilizó el entorno de desarrollo de Python para escribir, ejecutar y probar el programa. Durante este proceso, se corrigieron errores.
Se aprendió sobre el manejo de datos, algoritmos y diagramas de flujo. En este proyecto, se definieron variables para almacenar la longitud de la contraseña y las opciones seleccionadas por el usuario. También, se diseñó un algoritmo que describe paso a paso el proceso de generación de la contraseña. Esto se representó mediante un diagrama de flujo para facilitar la comprensión de la lógica del programa.
Las condicionales, como if, elif y else, permitieron verificar las opciones ingresadas por el usuario. Gracias a ellas, el programa determina qué tipos de caracteres deben incluirse en la contraseña. También se valida que existe al menos una opción seleccionada antes de generar el resultado.
Los bucles se utilizaron para repetir el proceso de creación de los caracteres que conforman la contraseña. Esto permite que el programa pueda generar nuevas contraseñas si el usuario lo desea. De esta manera, evita repetir código y hace que el sistema sea más eficiente.
Se emplearon estructuras de datos y funciones. En este caso, se usaron listas para almacenar los diferentes grupos de caracteres, como letras, números y símbolos. También, se implementaron funciones para organizar el código, dividir el programa en tareas específicas y facilitar su mantenimiento y reutilización.
